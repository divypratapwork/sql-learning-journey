### What I learned today
- Combining data from multiple tables using JOINs
- INNER JOIN to retrieve matching records
- LEFT JOIN and RIGHT JOIN to include unmatched rows
- FULL JOIN to retrieve all records from both tables
- CROSS JOIN to generate all possible combinations
- Identifying unmatched records using NULL conditions
- Writing multi-table joins for real-world business queries

### Learning method
- Hands-on SQL practice
- Solving real-world relational data problems

### Summary
This day focuses on mastering SQL joins, which are essential for working with relational databases and are heavily tested in interviews.
